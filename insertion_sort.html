El método de inserción directa, o ordenamiento por inserción, es un algoritmo sencillo que ordena elementos construyendo un subconjunto ordenado de forma iterativa. Comienza considerando el primer elemento como ordenado y, para cada elemento siguiente, lo compara con los ya ordenados, desplazando estos últimos hacia la derecha si son mayores. Luego, inserta el elemento en la posición correcta dentro del subconjunto. Este proceso se repite hasta que toda la lista esté ordenada. Es eficiente para listas pequeñas o casi ordenadas, con una complejidad de 𝑂(𝑛2) O(n2) en el peor caso y 𝑂(𝑛) O(n) en el mejor. Es un algoritmo estable, fácil de implementar y no requiere memoria adicional significativa, pero su rendimiento es limitado en grandes conjuntos de datos.
