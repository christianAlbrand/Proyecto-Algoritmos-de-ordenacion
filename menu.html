¿Qué son los algoritmos de ordenación?

Los algoritmos de ordenación son técnicas fundamentales en la informática que se utilizan para organizar los elementos de una estructura de datos, como un arreglo o una lista, en un orden específico, ya sea ascendente o descendente. Estos algoritmos son esenciales en una amplia variedad de aplicaciones, desde la búsqueda eficiente de datos hasta la optimización en procesos complejos.

Algoritmo Descripción Complejidad Temporal
Selection Sort Selecciona el elemento más pequeño y lo coloca en su posición final en cada iteración. O(n²)
Bubble Sort Compara pares de elementos adyacentes e intercambia si están en el orden incorrecto. O(n²)
Burbuja Mejorada Optimización del Bubble Sort que detiene el proceso si no hay intercambios en una pasada. O(n²) (mejor caso: O(n))
Merge Sort Divide el array en mitades, las ordena recursivamente y luego las combina. O(n log n)
Quick Sort Selecciona un pivote, organiza elementos menores y mayores alrededor de él. O(n log n) (peor caso: O(n²))
Heap Sort Usa un árbol binario para organizar los datos en un heap y ordenarlos. O(n log n)
Counting Sort Cuenta la cantidad de elementos en cada rango para ordenar rápidamente números pequeños. O(n + k)
Radix Sort Ordena números por sus dígitos, procesándolos de menor a mayor significancia. O(nk)
Bucket Sort Divide los elementos en varios "cubos" y luego los ordena individualmente. O(n + k)
